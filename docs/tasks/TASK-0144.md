## Formula
- Channel ROI % = (revenue - spend) / spend * 100
- ROAS = revenue / spend
- CAC = spend / new customers
- Payback days = CAC / (average check * daily visit frequency)
- Total ROI % = (total revenue - total spend) / total spend * 100
- Total CAC = total spend / total new customers
- LTV = average check * monthly visits * loyalty months
- LTV:CAC = LTV / total CAC
