## Method

The tool scores applicable criteria for the selected location type:

`score = collected weighted points / maximum applicable weighted points * 100`

Breakeven:

`gross margin % = 100 - (food cost % + royalty % + marketing %)`

`monthly breakeven sales = fixed costs / (gross margin % / 100)`

If gross margin is `<= 10%`, the point is shown as structurally weak.

## Verification Dataset

- Strong street point: all criteria `2`, default breakeven -> score near 100 and strong level.
- Weak street point: all criteria `0` -> score 0 and risky level.
- Unsustainable edge: food cost 70%, royalty 20%, marketing 15% -> gross margin below threshold and no breakeven sales.
- Existing site with rent/season/hour flags -> risk cards render.
