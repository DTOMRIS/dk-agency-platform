## Architecture
- 3-step form: restaurant profile → customer observations → AI persona generation
- DeepSeek `deepseek-chat` model, temperature 0.7, response_format: json_object
- 18-field persona JSON validated server-side (validatePersonaJSON)
- Cookie-based rate limit: 5 runs per 10 minutes
- localStorage: last 3 persona history (name + tagline + date)

## Persona Card Layout
- Left: profile (name, age, occupation, tagline, demographics)
- Right: insights (psychographics, digital, behavior, pain points)
- Bottom: marketing recommendations (message, channels, menu, do/don't)
