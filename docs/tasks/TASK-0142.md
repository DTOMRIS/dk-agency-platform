## Scope
- KALFA tier complaint analysis page
- Client-side instant keyword category matching
- DeepSeek server action with discovery questions
- Three result tabs: customer response, discovery questions, internal note
- CAPA block: investigation, closure criteria, corrective action, preventive action, recurrence check
- Copy button and localStorage recent analyses
- Four-language i18n

## Philosophy
The tool is not only a response writer. It helps the operator understand the complaint before replying. The central rule is that customers often remain unhappy because of how the issue is handled, not only because of the original issue.
