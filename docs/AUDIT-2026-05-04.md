## 4. KRİTİK PROBLEMLƏR (TƏCİLİ HƏLL)

### 🔴 KRİTİK-1: DEMO GİRİŞLƏR PRODUCTİON-DA GÖRÜNİR
- **Fayl:** `app/auth/login/page.tsx` (sətir 163-182)
- Açıq mətn: `member@dkagency.az / member123`, `admin@dkagency.az / admin123`, `dotomris@gmail.com / 12345678`
- **Həll:** production-da gizlə (`process.env.NODE_ENV !== 'production'`)
- **Müddət:** 15 dəq | Asan | KRİTİK

### 🔴 KRİTİK-2: ADMİN API ENDPOINT-LƏRİ QORUNMAYIB
- **Fayllar:** `app/api/admin/news/approve/route.ts`, `reject/route.ts`, `pending/route.ts`
- Heç bir auth check yoxdur — istənilən kəs news approve/reject edə bilər
- **Həll:** session check əlavə et (`session.plan === 'admin'`)
- **Müddət:** 30 dəq | Asan | KRİTİK

### 🔴 KRİTİK-3: DEFAULT JWT SECRET
- **Fayl:** `lib/auth/jwt.ts` (sətir 4)
- `JWT_SECRET || 'dk-agency-jwt-secret-change-in-production'`
- Əgər env var yoxdursa, insecure default istifadə olunur
- **Həll:** env var olmasa startup fail etsin
- **Müddət:** 15 dəq | Asan | KRİTİK

### 🔴 KRİTİK-4: HARDCODED PAROLLAR KOD-DA
- `lib/auth/mock-state.ts` — `'12345678'`, `'DkAdmin2026!'`, `'DkMember2026!'`
- `CLAUDE-DESIGN.md` (sətir 145) — `"admin123"` sənədlənib
- **Həll:** env var-a keçir, fallback sil
- **Müddət:** 30 dəq | Asan | KRİTİK

### 🟡 YÜKSƏK-5: RATE LİMİTİNG YOXDUR
Auth və AI endpoint-lərdə limit yoxdur:
- `/api/auth/login` — brute force açığı
- `/api/auth/register` — spam registration
- `/api/auth/forgot-password` — reset spam
- `/api/kazan-ai` — AI token consumption
- `/api/invoice-ocr` — image processing abuse
- **Həll:** IP-based rate limiter əlavə et
- **Müddət:** 2 saat | Orta | Yüksək

### 🟡 YÜKSƏK-6: console.log PRODUCTİON-DA (28 yerdə)
- 12+ API faylda `console.log`/`console.error`
- CLAUDE.md qaydası pozulub: "Do not use console.log in production code"
- **Həll:** sil və ya `NODE_ENV` guard
- **Müddət:** 1 saat | Asan | Orta
